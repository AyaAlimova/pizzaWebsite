Pizza Website

Overview
This project is a beautifully designed pizza website created using HTML and CSS. It includes three main pages: Home, Menu, and Order Online.

Pages
1. Home Page
Features:
Navigation bar with logo
Images with CSS animations (logo and main picture rotate on hover)
Description: The landing page with a navigation bar and a rotating logo and main image when hovered over.
2. Menu Page
Features:
Table displaying the menu
Images and styled text within the table
Description: A well-organized table with menu items, each having an image and descriptive text.
3. Order Online Page
Features:
Form elements including text fields, labels, checkboxes, radio buttons, dropdowns, and a submit button
Description: A form for customers to place orders, featuring various input elements for a user-friendly experience.
